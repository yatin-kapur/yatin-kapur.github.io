GAME THEORY (Module 8)

Definitions + Mini Examples

  • Dominant strategy: Best regardless of opponent. Ex: Defect in Prisoners' Dilemma
  • Dominated strategy: Always worse than another. Eliminate it, then re-check
  • Nash Eq (NE): Both play best responses; no one wants to deviate

Normal Form Example (Prisoners' Dilemma)

C (cooperate)D (defect)
C(-1, -1)(-9, 0)
D(0, -9)(-6, -6) NE
  • D is dominant for both. NE = (D,D) even though (C,C) is better for both.
  • Finding NE: For each column, underline Row's max. For each row, underline Col's max. NE = both underlined.

Sequential Game (Backward Induction)

You ─┬─ Invest ─── Rival ─┬─ Enter → (10, -2) │ └─ Stay out → (40, 0) ✓ └─ Don't ─── Rival ──┬─ Enter → (20, 10) ✓ └─ Stay out → (50, 0)
  • Solve from END: If Invest → Rival stays out (-2<0). If Don't → Rival enters (10>0).
  • You compare: Invest→40 vs Don't→20. Invest! (40>20)
  • Value of commitment = π(sequential) - π(simultaneous)

COMMITMENT (Module 11)

  • Must be visible, credible, irreversible. Ex: Cortés burns ships
  • Entry deterrence: Invest to make rival's entry payoff negative
  • Key insight: payoffs lower in ALL scenarios, but changes WHICH scenario happens
  • Preemption: product proliferation, excess capacity, predatory pricing

Entry Deterrence Pattern

Incumbent ─┬─ High capacity → Entrant stays out → (lower, 0) └─ Low capacity → Entrant enters → (even lower, +)
  • If π(high cap, no entry) > π(low cap, entry) → invest!

PRICE COMPETITION (Module 9)

Bertrand Trap

  • Identical goods, same MC → NE: P = MC, π = 0 (even with 2 firms!)
  • Ex: 4 firms, P=2000-Q, MC=0 → NE: P=0, π=0

Differentiated Bertrand

q₁ = a - p₁ + b·p₂, MC = c
π₁ = (p₁-c)(a-p₁+b·p₂)
FOC → p₁* = (a+c+b·p₂)/2
Symmetric NE: p* = (a+c)/(2-b)
Ex: q=5-p₁+0.5p₂, MC=1 → BR: p₁=3+0.25p₂ → NE: p*=$4. Monopolist would set $5.50.

COOPERATION (Module 10)

Cartels

  • Restrict output to quotas → monopoly price → split profits
  • Illegal (felony) in US, EU, Japan, Korea, Brazil
  • Each member tempted to cheat (produce more at high P)

Trigger Strategy & NPV

Trigger: "High P forever UNLESS someone deviates → P=MC forever" NPV(cooperate) = π_mono/n + (π_mono/n)/r = (π_mono/n)(1+1/r) NPV(deviate) = π_mono + 0 Cooperate when: r < 1/(n-1) 2 firms: r < 100% | 3 firms: r < 50% | 4 firms: r < 33%

Worked Example (from Final)

3 firms, MC=100, Q=1000-P, r=5%
Monopoly: MR=1000-2Q=100 → Q=450, P=550 π_mono = 450×450 = 202,500. Each: 67,500 NPV(coop) = 67,500 + 67,500/.05 = 1,417,500 NPV(dev) = 202,500 1.4M >> 202K → YES cooperate (also 5% < 50% ✓)

With Antitrust (detection prob X, penalty k×π)

E[cartel] = (1-X)·[π_c(1+1/r)] + X·[π_c - k·π_c] Cartel when E[cartel] > π_deviate → solve for X
Ex: 4 firms, π_mono=1M, r=20%, penalty=3×profits → X < 25%

Finite Repeated Games

  • Cooperation UNRAVELS — backward induction from last period
  • Period T: no future → deviate. Period T-1: T is deviate anyway → deviate...
  • Result: P=MC in ALL periods. Ex: 3-period game → P=0 every period

Tacit Collusion (Legal!)

  • No explicit agreement → not illegal. Trigger strategy is NE.
  • Easier: fewer firms, lower r, transparent prices, frequent interaction

MONOPOLY REVIEW FORMULAS

P = a - bQ → MR = a - 2bQ MR = MC → Q*, then P* = a - bQ* π = (P*-MC)·Q* MR = P(1 + 1/ε) Markup: (P-MC)/P = -1/ε CS = ½·Q·(P_max - P*) DWL = ½·(Qc-Qm)·(Pm-MC)
Ex: P=2000-Q, MC=0 → MR=2000-2Q=0 → Q=1000, P=$1000, π=$1M

Infinite Geometric Series

a + a/(1+r) + a/(1+r)² + ... = a·(1+r)/r = a + a/r (Current period + PV of all future periods)

ASYMMETRIC INFO (Module 12)

Adverse Selection (Hidden Type — BEFORE transaction)

  • Informed knows quality; uninformed doesn't
  • Lemons spiral: Offer avg price → good sellers exit → avg drops → more exit → only worst quality remains
Qualities: q=0 (bad) ... q=1 (good) Buyer offers p(avg=0.5) → q>0.5 exit → offers p(avg=0.25) → q>0.25 exit → ... → only q=0 remains = FAILURE
Ex: Used cars, health insurance (sick buy, healthy don't → premiums rise → spiral)

Moral Hazard (Hidden Action — AFTER transaction)

  • Unobserved risky behavior after deal is made
  • Ex: Reckless driving after insurance, banks "too big to fail", employee shirking
  • Fixes: monitoring, incentive pay, stock options, deductibles, no-claim bonus

Signaling vs Screening

PLATFORMS & NETWORKS (Module 13)

Two Types of Network Effects

Platform Pricing

Subsidize elastic/scarce side, charge inelastic side AmEx: merchants +3%, cardholders -1%, net = 2% $100 purchase: merchant pays $103, buyer pays $99

Winner-Take-Most

  • Network effects + incompatibility → market tips to one platform
  • Critical mass: min users for self-sustaining growth. Ex: VHS beat Betamax
  • Why "most" not "all"? Multi-homing, niches, regulation
  • Interoperability → coexistence (email). No interop → tipping (consoles)
  • TS-maximizing price can be below MC (network externality justifies subsidy)

EXAM CHECKLIST

  • □ Matrix? → dominated strats → best responses → NE
  • □ Game tree? → backward induction from end
  • □ Repeated? → NPV: coop vs deviate. r < 1/(n-1)?
  • □ Finite repeat? → unravels, NE every period
  • □ Bertrand? → P=MC unless escape (5 ways)
  • □ Commitment? → compare invest+no entry vs no invest+entry
  • □ Info problem? → adverse selection (before) vs moral hazard (after)
  • □ Signal vs screen? → who acts: informed or uninformed?
  • □ Network effects? → direct vs indirect, interop vs not
  • □ Always: MR = a-2bQ, MR=MC, π=(P-MC)Q
