Game Theory (Mod 8)

Definitions

  • Dominant strategy: best regardless of opponent. Ex: Defect in PD
  • Dominated strategy: always worse than another. Eliminate, re-check.
  • Nash Eq (NE): both play best responses; no one wants to deviate.

Sequential (backward induction)

You ─┬─ Invest ─── Rival ─┬─ Enter → (10, −2) │ └─ Stay → (40, 0) ✓ └─ Don't ─── Rival ──┬─ Enter → (20, 10) ✓ └─ Stay → (50, 0)
  • Solve from end. If Invest → Rival stays (−2<0). If Don't → Rival enters (10>0).
  • You: Invest→40 vs Don't→20. Invest!
  • Value of commitment = π(seq) − π(sim).

Commitment (Mod 11)

  • Must be visible, credible, irreversible. Cortés burns ships.
  • Entry deterrence: invest to make rival's entry payoff negative.
  • Key: payoffs lower in ALL scenarios, but changes WHICH scenario happens.
  • Preemption: product proliferation, excess capacity, predatory pricing.

Entry deterrence pattern

Incumbent ─┬─ High cap → Entrant out → (lower, 0) └─ Low cap → Entrant in → (even lower, +)
  • If π(high cap, no entry) > π(low cap, entry) → invest.

Price Competition (Mod 9)

Bertrand trap

  • Identical goods, same MC → NE: P = MC, π = 0 (even with 2 firms).
  • Ex: 4 firms, P=2000−Q, MC=0 → NE: P=0, π=0

Differentiated Bertrand

q₁ = a − p₁ + b·p₂, MC = c π₁ = (p₁−c)(a−p₁+b·p₂) FOC → p₁* = (a+c+b·p₂)/2 Symmetric NE: p* = (a+c)/(2−b)
Ex: q=5−p₁+0.5p₂, MC=1 → BR: p₁=3+0.25p₂ → NE: p*=$4. Monopolist: $5.50.

Cooperation (Mod 10)

Cartels

  • Restrict output to quotas → monopoly price → split profits.
  • Illegal (felony) in US, EU, Japan, Korea, Brazil.
  • Each member tempted to cheat (more output at high P).

Trigger strategy & NPV

Trigger: "High P forever UNLESS deviation → P=MC forever" NPV(coop) = π_mono/n + (π_mono/n)/r = (π_mono/n)(1+1/r) NPV(deviate) = π_mono + 0 Cooperate when: r < 1/(n−1) 2 firms: r<100% | 3: r<50% | 4: r<33%

Worked example (from Final)

3 firms, MC=100, Q=1000−P, r=5%
Mono: MR=1000−2Q=100 → Q=450, P=550 π_mono = 450×450 = 202,500. Each: 67,500 NPV(coop) = 67,500 + 67,500/.05 = 1,417,500 NPV(dev) = 202,500 1.4M >> 202K → YES cooperate (5% < 50% ✓)

With antitrust (X = detect prob, k = penalty)

E[cartel] = (1−X)·[π_c(1+1/r)] + X·[π_c − k·π_c] Cartel when E[cartel] > π_dev → solve for X
Ex: 4 firms, π_mono=1M, r=20%, k=3 → X < 25%

Finite repeated games

  • Cooperation UNRAVELS - backward induction from last period.
  • Period T: no future → deviate. Period T−1: T deviate anyway → deviate…
  • Result: P=MC in ALL periods. 3-period game → P=0 every period.

Tacit collusion (legal)

  • No explicit agreement → not illegal. Trigger strategy is NE.
  • Easier: fewer firms, lower r, transparent prices, frequent interaction.

Monopoly Review

P = a − bQ → MR = a − 2bQ MR = MC → Q*, then P* = a − bQ* π = (P*−MC)·Q* MR = P(1 + 1/ε) Markup: (P−MC)/P = −1/ε CS = ½·Q·(P_max − P*) DWL = ½·(Qc−Qm)·(Pm−MC)
Ex: P=2000−Q, MC=0 → MR=0 → Q=1000, P=$1000, π=$1M

Infinite geometric series

a + a/(1+r) + a/(1+r)² + ... = a·(1+r)/r = a + a/r (current period + PV of all future periods)

Asymmetric Info (Mod 12)

Adverse selection (hidden type - BEFORE)

  • Informed knows quality; uninformed doesn't.
  • Lemons spiral: offer avg price → good sellers exit → avg drops → more exit → only worst remains.
Qualities: q=0 (bad) ... q=1 (good) Buyer offers p(avg=0.5) → q>0.5 exit → p(avg=0.25) → q>0.25 exit → ... → only q=0 remains = FAILURE
Ex: used cars; health insurance (sick buy, healthy don't → spiral)

Moral hazard (hidden action - AFTER)

  • Unobserved risky behavior after deal is made.
  • Ex: reckless driving after insurance, "too big to fail", shirking
  • Fixes: monitoring, incentive pay, stock options, deductibles, no-claim bonus.

Platform pricing

Subsidize elastic/scarce side, charge inelastic side AmEx: merchants +3%, cardholders −1%, net = 2% $100 purchase: merchant pays $103, buyer pays $99

Winner-take-most

  • Network effects + incompatibility → market tips to one platform.
  • Critical mass: min users for self-sustaining growth. VHS beat Betamax.
  • Why "most" not "all"? Multi-homing, niches, regulation.
  • Interop → coexistence (email). No interop → tipping (consoles).
  • TS-max price can be below MC (network externality justifies subsidy).

Exam Checklist

  • □ Matrix? → dominated strats → best responses → NE
  • □ Game tree? → backward induction from end
  • □ Repeated? → NPV: coop vs deviate. r < 1/(n−1)?
  • □ Finite repeat? → unravels, NE every period
  • □ Bertrand? → P=MC unless escape (5 ways)
  • □ Commitment? → compare invest+no entry vs no invest+entry
  • □ Info problem? → adverse selection (before) vs moral hazard (after)
  • □ Signal vs screen? → who acts: informed or uninformed?
  • □ Network effects? → direct vs indirect, interop vs not
  • □ Always: MR = a−2bQ, MR=MC, π=(P−MC)Q
